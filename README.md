<h1>3D 프린팅 도면 제출 폼 프로젝트</h1>
URL: https://github.com/hyerin10/jsp_3D_drawing_submission_form

<h3>핵심 기능</h3>
<ul>
  <li>도면 견적 제출(입력), 수정, 삭제</li>
  <li>도면 견적 게시판</li>
  <li>도면 견적 페이지</li>
</ul>
<h3>사용한 기술</h3>
<ul>
  <li>Java</li>
  <li>JSP</li>
  <li>Spring 프레임워크 사용</li>
  <li>HTML</li>
  <li>CSS</li>
  <li>MySQL</li>
  <li>JDBC 사용</li>
  <li>JSON형태로 출력</li>
  <li>Github 사용</li>
</ul>
